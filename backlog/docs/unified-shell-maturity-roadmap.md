# Unified Shell Maturity Roadmap

The canonical Unified Shell maturity tracker lives at:

`Docs/superpowers/trackers/unified-shell-maturity-roadmap.md`

Durable QA evidence lives at:

`Docs/superpowers/qa/unified-shell/`

Backlog tasks in `backlog/tasks/` are PR-sized execution units. Parent tasks represent maturity phases; child tasks represent implementation or QA slices.

Do not mark a task or phase complete because UI renders or a button is clickable. Completion requires automated evidence, running-app QA evidence where product behavior is in scope, and a repo-tracked QA summary.
