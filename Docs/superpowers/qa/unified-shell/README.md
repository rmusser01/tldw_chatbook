# Unified Shell QA Evidence

This directory stores durable QA summaries for Unified Shell maturity work.

Large screenshots, videos, temporary probe files, and logs may live outside git, but each task or phase must include a concise markdown summary here that records what was verified, what failed, and what remains uncertain.

## Phase Index

- `phase-0/` - Canonical tracking setup evidence.
- `phase-1/` - Shell contract and destination affordance walkthroughs.
- `phase-2/` - Home dashboard and active-work control walkthroughs.
- `phase-3/` - Console live-work hub walkthroughs.
- `phase-4/` - Destination service adoption walkthroughs.
- `phase-5/` - Capability-state and recovery walkthroughs.
- `phase-6/` - First-time user, power-user, and Nielsen heuristic closeout.

## Required Summary Fields

- Date
- Branch and commit
- Task or phase
- Environment
- Entry path
- Steps attempted
- Visual usability notes
- Keyboard path result
- Mouse/click path result
- Functional result
- Defect severity
- Evidence links or artifact paths
- Residual risk
